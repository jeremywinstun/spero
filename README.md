# spero
This is the ROS package (Noetic) for SPERO. SPERO is a robot developed by Petra Christian University in response to the COVID-19 Pandemic
SPERO is a robot that is able to disinfect rooms using UV-C radiation automatically with a magnetic line following the method 
SPERO is built based on ROS Noetic and is also equipped with MQTT

For the User Interface download here: https://github.com/Jeremywinst/uispero
